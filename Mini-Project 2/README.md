### üìò HW2 - Question 6: Static & Adaptive RBF Networks
* **Objective:** Design and implement Static and Adaptive Radial Basis Function (RBF) Neural Networks.
* **Key Implementations:**
    * **Static RBFNN:** Fixed-structure network trained using Linear Least Squares (LLS).
    * **Adaptive RBFNN:** Dynamic network inspired by the **M-RAN** algorithm that automatically adjusts its topology by adding/pruning neurons during training.
* **Tools:** Python, NumPy, Matplotlib.
